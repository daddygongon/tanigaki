$F_2$上の線形符号の生成行列を考える．

2 gm_sys, gm_ns

今，$K=4, N=8$とする． gm(generate matrix, 生成行列)を ns_to_sys, nsをsysにする 生成行列の中で， \[ G_{sys}=(I_4\, P) \] であるものを既約標準形という. ここで，$I_4$は$4$次単位行列， $P$は\(K \times (N-K)=4\times4\) 行列である．

既約標準形(irred:Irreducible standard type) 

gm_ns = 
01100010
11010111
00111001
10101011

谷垣のコードの結果
10000010
01010111
00111001
00001011

理想
gm_sys = i行-j列成分
1000 0010
0100 0111
0010 1001
0001 1011

を考える．

手順としては，

  • \(i=0\) とする.
  • 生成行列の (i, i) 成分 \(g_{i,i}\) を調べる
    • \(g_{i,i} = 1\) であるなら，基本行操作を行う
    • \(g_{i,i} = 0\) であるなら，
      • \(g_{j,i} = 1\) であるような \(j\) を探し(j > i)，
      • 第 \(i\) 行と第 \(j\) 行の成分 を交換する.
      • その上で，前項と同様，基本行操作を行う
  • $i = K − $1 なら終了.
    • それ以外の場合は，
    • i をイ ンクリメントして，手順 (b) へ戻る.

とされていた．

この中で，(b)-iiの二つ目の手順で

  • 第 \(i\) 行と第 \(j\) 行の成分 を交換する.

とあるが，最初のコードでは， 

if(gm_ns[j][i]==1)
{
	tmp=gm_ns[j][i];
	gm_ns[j][i]=gm_ns[i][i];
	gm_ns[i][i]=tmp;
}

としており，これでは行列のi-j成分を入れ替えているだけであった．

そこで，新たにi,j行を入れ替える関数(swap_i_j)を以下の通り作成した． 

void swap_i_j
(
        unsigned int gm_ns[K][N],
        unsigned int i,
        unsigned int j
)
{
        unsigned int temp;
        int k;

        for(k=0;k<N;k++){
                temp=gm_ns[i][k];
                gm_ns[i][k]=gm_ns[j][k];
                gm_ns[j][k]=temp;
        }
}
